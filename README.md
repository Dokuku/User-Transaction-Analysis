# User-Transaction-Analysis
Сравнил пользовательские транзакции QR и Card платежей

Таблица состоит из транзакций от юр. лиц. В ней указаны их id (merchant_id), сегмент мерчанта, город, способ оплаты и подтвержден ли заказ (is_success)

Инструменты: Pandas, matplotlib, seaborn

Вывод: Были найдены возможные статистическе выбросы, а также этапы потери конверсии
